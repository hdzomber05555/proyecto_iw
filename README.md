# Gestor de Inventario

Sistema sencillo para controlar productos, existencias y movimientos en tiempo real.

## Características

- Iniciar sesión con usuario y contraseña.

- Ver un listado de ítems del inventario con paginación y buscador.

- Crear, editar, ver y borrar ítems.

- Guardar una preferencia de tema (claro/oscuro) en una cookie.

- Proteger los formularios con CSRF y validar en servidor.

- Hacer el borrado dentro de una transacción con rollback y auditoría.

## Tecnologías
- PHP

- MYSQL

- HTML

- CSS
